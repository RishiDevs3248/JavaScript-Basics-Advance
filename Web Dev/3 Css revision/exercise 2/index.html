Designing for Readability

By Aarav Mehta · Sep 1, 2025

When we read on screens, details like line-height, measure, and contrast dramatically affect comprehension and fatigue.

Good typography is not just aesthetic—it's functional . It guides the eye, clarifies hierarchy, and improves retention.

“Typography is the craft of endowing human language with a durable visual form.” — Robert Bringhurst

Core Practices

  • Use a comfortable line-height and readable font-size.
  • Limit line length (measure) to 60–75 characters.
  • Establish consistent spacing between blocks.

Inline code like line-height: 1.6; and letter-spacing can be used to fine-tune text rhythm.
